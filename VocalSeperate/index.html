使用spleeter提取人声。

简易方法（GUI）（使用GPU提速请按照下面复杂的方法配置）

挂梯子在这里下载

https://1drv.ms/u/s!AhbM98WiPUJYmSsVUi8LeNVbf14a?e=LBOgP9

或通过微云下载 链接: https://share.weiyun.com/54TVaAi

安装包并安装msi。

手动挂梯子下载https://github.com/deezer/spleeter/releases

下的 xstems.tar.gz，（x=2,4,5) 或使用2020.3.15更新的国内下载源：https://gitee.com/flottant/spleeterdownload/repository/archive/master.zip

使用IDM下载会出错，请使用浏览器自带下载。分卷解压缩，放置在pretrained_models（用everything找）文件夹下，并将之前文件夹中内容删除。

pretrained_models文件夹下应该有2stems，4stems，5stems三个子文件夹。

然后找到桌面图标运行即可。运行报错通常是因为电脑性能带不动，尤其是内存。（建议 8GB+）

配置anconda

下载与安装

在这里下载：https://repo.anaconda.com/archive/Anaconda3-2020.02-Windows-x86_64.exe
如果有新版本，在这里下载：https://www.anaconda.com/distribution/
页面拉到中间，选择python3.7 version
安装时候遇到打勾，请把两个勾都打上（第一个是安装到PATH），并选择为所有人安装。
安装完后通过在命令行中输入conda来检查。

配置镜像源

执行

conda config –set show_channel_urls yes

然后找到.condarc文件（请使用everything搜索）

用文本编辑工具打开

删除所有内容，添加

channels:
  - defaults
show_channel_urls: true
channel_alias: https://mirrors.tuna.tsinghua.edu.cn/anaconda
default_channels:
  - https://mirrors.tuna.tsinghua.edu.cn/anaconda/pkgs/main
  - https://mirrors.tuna.tsinghua.edu.cn/anaconda/pkgs/free
  - https://mirrors.tuna.tsinghua.edu.cn/anaconda/pkgs/r
  - https://mirrors.tuna.tsinghua.edu.cn/anaconda/pkgs/pro
  - https://mirrors.tuna.tsinghua.edu.cn/anaconda/pkgs/msys2
custom_channels:
  conda-forge: https://mirrors.tuna.tsinghua.edu.cn/anaconda/cloud
  msys2: https://mirrors.tuna.tsinghua.edu.cn/anaconda/cloud
  bioconda: https://mirrors.tuna.tsinghua.edu.cn/anaconda/cloud
  menpo: https://mirrors.tuna.tsinghua.edu.cn/anaconda/cloud
  pytorch: https://mirrors.tuna.tsinghua.edu.cn/anaconda/cloud
  simpleitk: https://mirrors.tuna.tsinghua.edu.cn/anaconda/cloud

保存退出。

配置CUDA

下载与安装

不使用显卡版本的请跳过。

在这里下载，https://developer.nvidia.com/cuda-downloads

选择好与系统对应的最新版本，一路next安装完毕。

配置

定位到安装路径下的Anaconda3\Library\bin文件夹， 把此文件夹里的 mkl_*.dll 文件也就是mkl_开头的DLL文件全部复制粘贴到系统分区windows下的system32路径里，如 ‘C:\Windows\System32’ 路径下，如果提示是否覆盖，选择是即可。
安装路径里Anaconda3\Library\bin文件夹里找到libiomp5md.dll，把此文件复制粘贴到系统分区windows下的system32路径里，如 ‘C:\Windows\System32’ 路径下即可。

配置spleeter

conda环境设置

依次执行

conda create -n python36 python=3.6

conda activate python36

conda config –add channels conda-forge

conda config –set channel_priority flexible

安装spleeter

conda install -c conda-forge spleeter-gpu 或 conda install -c conda-forge spleeter #CPU版本

spleeter separate -i 一个mp3音频绝对路径 -o audio_output

这时候会提示正在从github下载训练好的模型，直接关闭退出。

手动挂梯子下载https://github.com/deezer/spleeter/releases

下的xstems.tar.gz，（x=2,4,5) 或使用2020.3.15更新的国内下载源：https://gitee.com/flottant/spleeterdownload/repository/archive/master.zip

使用IDM下载会出错，请使用浏览器自带下载。分卷解压缩，放置在pretrained_models（用everything找）文件夹下，并将之前文件夹中内容删除。

pretrained_models文件夹下应该有2stems，4stems，5stems三个子文件夹。

找到anconda prompt(anconda), 运行输入指令

conda activate python36

spleeter separate -i 一个mp3音频绝对路径 -o audio_output

使用everything找到audio_output这个文件夹，检查里面的vocals.wav是否播放正常。

使用spleeter

参考https://github.com/deezer/spleeter/wiki/2.-Getting-started

如果仅仅提取人声的话使用

spleeter separate -i 一个mp3音频绝对路径 -o audio_output

这个格式。

注：everything官网https://www.voidtools.com/zh-cn/

请使用安装版，安装时一路next即可。
